🪙 How Toonie Tasks work
  • Plan these at Sunday Planning — they're the weekly $2 earners, an extra on top of her routines.
  • Each window's tasks unlock only after Violet finishes that routine within its time window.
  • She can earn each task once per window; the list renews at the next window.
  • Set the $ value per task (defaults to $2). Always hit Save when done.
