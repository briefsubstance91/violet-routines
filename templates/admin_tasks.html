📋 How Tasks work

These are the checklist items Violet sees inside each routine. Changes here update the app instantly after saving.

  • Click any task text to edit it inline
  • Drag the ⠿ handle to reorder tasks within a routine
  • Tap + Add sub-task under a task to nest a step beneath it (e.g. "Socks" under "Get dressed")
  • Add category tags (e.g. hygiene, school) with the + tag field — type and press Enter
  • Hit × to delete a task or sub-task, + Add task to create a new task
  • Always click Save when you're done — unsaved changes are lost on refresh
