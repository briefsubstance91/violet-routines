✨ How Events work

These show up in the Today's Extras card on Violet's home page when they're due.

  • When — recurring: daily, weekdays, weekends, a day list like Mon,Wed,Fri, or monthly:15. One-off: a date like 2026-06-25.
  • Type — task shows a checkbox Violet can tick off; event is a friendly heads-up only.
  • Time (optional) — like 4:00 PM; if set, a reminder is pushed at that time on days it's due.
  • Past one-off dates are tidied up automatically. Click Save when done.
